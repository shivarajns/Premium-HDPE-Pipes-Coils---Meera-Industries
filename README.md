## Project Overview
This project is a pixel-perfect implementation of the provided Figma design, focusing on UI accuracy, responsiveness, and interactive elements.

The objective was to build a fully functional and visually consistent web page without using any external frameworks or libraries.

---

## Features

- Pixel-perfect implementation based on Figma design  
- Fully responsive layout (desktop, tablet, and mobile)  
- Sticky header  
  - Appears after scrolling beyond the first fold  
  - Smooth show and hide transitions  
- Interactive image carousel  
  - Smooth sliding functionality  
  - Hover-based zoom preview effect  
- Modern CSS layout using Flexbox and Grid  
- Clean and well-structured code  
- Cross-browser compatibility  

---

## Tech Stack

- HTML5 – Semantic structure  
- CSS3 – Styling, layout, and animations  
- JavaScript (Vanilla) – Interactivity and DOM manipulation
